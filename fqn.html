What is the Document Object Model (DOM)?

The Document Object Model is a programming interface for web documents. It represents the structure of a webpage as a tree of objects, allowing programs and scripts to dynamically access, modify, and manipulate the content, structure, and style of the document.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in JavaScript that allows you to manage events efficiently by taking advantage of the way events propagate through the DOM. Instead of adding event listeners to multiple child elements, you add a single event listener to a parent element. This listener can then respond to events that occur on any of its children.
